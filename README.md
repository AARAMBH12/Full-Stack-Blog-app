# Blog App

A simple, full-stack blog application where users can read, write, edit, and delete blog posts. This application features basic CRUD operations (Create, Read, Update, Delete) and allows users to interact with posts via a user-friendly interface.

## Features

- User Authentication (Login/Signup)
- Create, Edit, and Delete Posts
- View All Blog Posts
- Post Details Page
- Search and Filter Posts by Categories/Tags
- Comment on Posts


## Tech Stack

- **Frontend**: React.js, Redux, Bootstrap (or Material UI for styling)
- **Backend**: Node.js, Express.js
- **Database**: MongoDB (with Mongoose)
- **Authentication**: JWT (JSON Web Token)
